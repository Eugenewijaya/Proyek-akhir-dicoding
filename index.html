Tentang Saya

Halo, saya Evid Wijaya, mahasiswa semester 5 Program Studi Sistem Informasi di STMIK KUWERA. Saat ini, saya menjabat sebagai Ketua BEM di School of Technopreneur Nusantara.

Pendidikan

Saya saat ini menempuh studi di Program Studi Sistem Informasi di STMIK KUWERA dengan IPK 3.81/4.00. Fokus saya termasuk Manajemen Bisnis, Sistem Informasi Manajemen, dan Web Development.

Pengalaman

Saya memiliki pengalaman dalam administrasi umum dan fotografi. Salah satu pengalaman berharga saya adalah terlibat dalam Kongres Luar Biasa HIMPSI 2024, di mana saya membuat sistem pendaftaran dan e-voting.

Keterampilan

Saya mahir dalam menggunakan perangkat lunak seperti Microsoft Office, Canva, Google Workspace, dan Meta Business Suite. Selain itu, saya terampil dalam SEO, menulis, dan fotografi dengan kamera DSLR.

Fokus Saat Ini

Saat ini, saya memfokuskan diri pada pengembangan keterampilan dalam cloud computing dan membangun portofolio saya dari berbagai acar volunteer.
